# Contribuindo para o Mulinco Linktree

Este é um projeto de portfólio pessoal e open-source destinado a compartilhar links e projetos. Sinta-se à vontade para usá-lo como inspiração para o seu próprio linktree pessoal!

## Como Contribuir

Embora este seja um projeto pessoal, sugestões e melhorias são sempre bem-vindas, especialmente em termos de acessibilidade, performance e boas práticas de código.

Se você deseja contribuir:

1. Faça um Fork do repositório.
2. Crie uma branch para a sua feature ou correção (`git checkout -b feature/minha-feature`).
3. Commit suas mudanças usando [Conventional Commits](https://www.conventionalcommits.org/) (`git commit -m 'feat: minha nova feature'`).
4. Faça o Push para a branch (`git push origin feature/minha-feature`).
5. Abra um Pull Request explicando detalhadamente o que foi alterado e por quê.

## Rodando Localmente

1. Clone o repositório.
2. Instale as dependências com `pnpm install` (este projeto usa `pnpm`).
3. Rode o servidor de desenvolvimento com `pnpm run dev`.
4. Para rodar os testes, utilize `pnpm run test`.
5. Acesse `http://localhost:3000`.

Obrigado pelo interesse! ✦
